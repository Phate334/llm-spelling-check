Risk threshold

數值越低越容易把字標成 suspicious，recall 通常會變高，但 false positive 也會增加；數值越高越保守，可能少誤抓但也更容易漏掉錯字。

Results

Metrics

P/R/F1 分別是 Precision、Recall、F1。Precision 看「抓到的有多準」，Recall 看「該抓的有沒有抓到」，F1 是兩者的平衡分數。

Detection Precision
被標成 suspicious 的位置中，有多少真的是錯字；越高代表少誤抓。
Detection Recall
所有真正錯字中，有多少被標出來；越高代表少漏抓。
Detection F1
Detection Precision 和 Detection Recall 的綜合分數，用來看偵測整體平衡。
Correction Precision
模型實際修改的位置中，有多少真的改對；越高代表少改錯。
Correction Recall
所有真正錯字中，有多少最後被改成正確答案；越高代表修正覆蓋較完整。
Correction F1
Correction Precision 和 Correction Recall 的綜合分數，用來看修正整體平衡。
FPR
False Positive Rate，正常字被誤標成 suspicious 的比例；越低越好。
