LLM Spelling Check

Input

Risk threshold

數值越低越容易把字標成 suspicious，recall 通常會變高，但 false positive 也會增加；數值越高越保守，可能少誤抓但也更容易漏掉錯字。

Results

Metrics
Detection Precision
被標成錯字的位置中，有多少真的錯。
Detection Recall
所有真正錯字中，有多少被找出來。
Detection F1
Precision 和 Recall 的綜合分數。
Correction Precision
模型實際修改的位置中，有多少改成正確答案。
Correction Recall
所有真正錯字中，有多少被改成正確答案。
FPR
正常字被誤標成 suspicious 的比例。
